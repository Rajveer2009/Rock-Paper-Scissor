# Rock-Paper-Scissors

A modern Rock Paper Scissors game that uses AI-powered hand gesture recognition. Play against the computer using real hand gestures captured through your webcam!

## Features

- 🎮 **Hand Gesture Recognition** - Uses MediaPipe AI to detect Rock, Paper, Scissors gestures
- 📷 **Webcam Integration** - Real-time gesture detection through your camera
- 🏆 **Score Tracking** - Keep track of wins, losses, and ties
- 👍 **Thumbs Up to Start** - Quick game start with thumbs up gesture

## How to Play

1. Allow camera permissions when prompted
2. Click "Start Game" or show thumbs up (👍) gesture
3. Make your gesture during countdown:
   - ✊ = **Rock**
   - 🖐️ = **Paper**
   - ✌️ = **Scissors**
